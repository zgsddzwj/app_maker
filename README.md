# App Store 截图生成器

一个简单易用的 App Store 截图生成工具，帮助开发者快速创建符合 App Store 要求的宣传截图。

## 功能特点

### 📱 多设备支持
- iPhone 6.7" (1290 x 2796)
- iPhone 6.5" (1242 x 2688)
- iPhone 5.5" (1242 x 2208)
- iPad 13" (2048 x 2732)
- iPad 11" (1668 x 2388)

### 🖼️ 截图上传
- 支持点击上传或拖拽上传
- 支持 PNG、JPG 格式
- 可随时移除已上传的截图

### ✍️ 文字编辑
- 自定义标题和副标题
- 可调整文字颜色
- 可调整字体大小

### 🎨 样式定制
- **背景类型**：
  - 渐变背景（可自定义颜色或选择预设）
  - 纯色背景
  - 图片背景
- **手机框架**：可开关，可自定义颜色

### 🔧 交互功能
- **拖拽调整位置**：可直接在预览区拖动文字和截图，自由调整位置
- **实时预览**：所有修改即时显示
- **缩放控制**：调整预览大小

### 📤 导出功能
- 一键导出 PNG 格式高清截图
- 文件名包含设备类型和时间戳

## 使用方法

1. 选择目标设备尺寸
2. 上传你的 App 截图
3. 输入宣传标题和副标题
4. 自定义背景和文字样式
5. 拖动调整元素位置
6. 点击"导出截图"下载成品

## 本地运行

直接在浏览器中打开 `index.html` 文件即可使用。

```bash
# 克隆项目
git clone https://github.com/zgsddzwj/app_maker.git

# 打开页面
open index.html
```

## 技术栈

- HTML5
- CSS3
- Vanilla JavaScript
- Canvas API (用于导出图片)
